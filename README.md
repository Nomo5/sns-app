# メッセージ機能付き簡易SNSアプリ
phpを使って簡易的なSNSアプリを作成しました。

## 使用感
[![紹介動画]((https://user-images.githubusercontent.com/106419953/184296320-3d2695cc-ef84-4ac7-b7f4-351b14815bc3.png))](https://youtu.be/mLiqK0H1wOs)

## 使用技術/動作環境
* HTML/CSS
* JavaScript
* PHP 8.1.6
* XAMPP 8.1.6
* Apache/2.4.53
* MariaDB 10.4.24
* Google Chrome

## 機能
* ユーザーの新規登録
* ログイン・ログアウト
* 投稿、投稿の閲覧、投稿の削除
* いいね機能
* 他ユーザーとのチャット機能
